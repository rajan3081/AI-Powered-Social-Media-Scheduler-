# AI-Powered Social Media Scheduler

This repository contains a full-stack Social Media Scheduler application with AI features.

Structure:
- `backend/` - Node.js/Express server, routes, models, and scheduler
- `frontend/` - React (Vite) app for the UI

Quick start

Prerequisites:
- Node.js (>=16)
- npm or yarn

Run backend:

```bash
cd backend
npm install
npm start
```

Run frontend:

```bash
cd frontend
npm install
npm run dev
```
